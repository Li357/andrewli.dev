Understanding quotient groups and normal subgroups, intuitively

Jan 2021

One of the defining characteristics of math is breaking down complex objects into simpler pieces that are easier to study. For instance, we prime-factorize integers and decompose matrices, and in the process gain a better understanding of the very nature of our original objects. In group theory where the objects of study are groups, this question becomes: how do we break down complicated groups into more wieldy ones?

Exploration

After learning about Lagrange’s theorem and how subgroups partition groups into the original subgroup and its cosets, a natural idea might be to leverage this to break down groups. For example, take the subgroup H={0,2}H = \{0, 2\}H={0,2} of Z4\mathbb Z_4Z4​ and its coset 1+H={1,3}1 + H = \{1, 3\}1+H={1,3}. By reorganizing the Cayley tables, we might end up with something like this:

If you squint, you might see something that resembles the Cayley table of Z2\mathbb Z_2Z2​, where instead of the integers 000 and 111 we have the cosets 0+H=H0 + H = H0+H=H and 1+H1 + H1+H. And just based off the clumped Cayley table, there’s a sensible operation ∗*∗ to “combine” these cosets to give another coset:

(a+H)∗(b+H)={x+y:x∈a+H,y∈b+H}=(a+b)+H. \begin{aligned} (a+H) * (b+H) &= \{x + y : x \in a+H, y \in b+H\} \\ &= (a+b) + H. \end{aligned} (a+H)∗(b+H)​={x+y:x∈a+H,y∈b+H}=(a+b)+H.​

For instance, H∗(1+H)={0+1,0+3,2+1,2+3}={1,3}=1+HH * (1+H) = \{0 + 1, 0 + 3, 2 + 1, 2 + 3\} = \{1, 3\} = 1+HH∗(1+H)={0+1,0+3,2+1,2+3}={1,3}=1+H. And it even seems to work when we choose a different coset representative 2+H=H2 + H = H2+H=H, since (2+H)∗(1+H)=1+H(2+H) * (1+H) = 1+H(2+H)∗(1+H)=1+H as well. It’s a great exercise to verify this works for any combination of HHH and 1+H1+H1+H, and with different coset representatives. So now we’ve got a set of two elements, {H,1+H}\{H, 1+H\}{H,1+H} that has a closed operation. It looks like a group and smells like one1, one smaller than the group Z4\mathbb Z_4Z4​ of four elements we started with!

So again the set {H,(12)H}\{H, (12)H\}{H,(12)H} is equipped with a closed operation ∗*∗, making it a very convincing group3 and three times smaller than our original S3S_3S3​! You might start to wonder if this process works for all groups. That’s until you try a second nonabelian example for good measure, with H={(),(12)}H = \{(), (12)\}H={(),(12)} of S3S_3S3​ and its cosets (123)H={(123),(13)}(123)H = \{(123), (13)\}(123)H={(123),(13)} and (132)H={(132),(23)}(132)H = \{(132), (23)\}(132)H={(132),(23)}:

The next question then becomes: what made the first two subgroups special?

Building Intuition

Before we answer that question, I want to present some intuition that’ll guide us in arriving at the answer–an answer that always felt a little arbitrary to me otherwise.

First of all, what does it mean for two elements to be in the same coset? Well, elements x,yx, yx,y land in the same coset yHyHyH exactly when x=yhfor some h∈H.x = yh \quad\quad\text{for some } h \in H.x=yhfor some h∈H. On an intuitive level, you might interpret this as xxx is “yyy-like”. In our original group, we would have differentiated between xxx and yyy. But once placed into the same coset, we agree that xxx is “just enough like” yyy such that they end up in the same coset. And that loss of differentiation is a tradeoff: we lose some detail of our original group but end with a smaller one–like lowering the resolution on an image.

To solidify this intuition, let’s define “yyy-like.”

DEFINITION. An element xxx is said to be “yyy-like” if and only if x=yhx = yhx=yh for some h∈Hh \in Hh∈H.

Now let’s return to aH∗bH=abHaH * bH = abHaH∗bH=abH. This was our previous requirement to create a smaller group from the cosets, and with this new definition, it should feel natural. We’d expect an element of aH∗bHaH * bHaH∗bH that looks like an aaa-like element multiplied by a bbb-like element to produce an ababab-like one in abHabHabH. Conversely, we’d expect an ababab-like element in abHabHabH to be the product of an aaa-like and bbb-like element in aH∗bHaH * bHaH∗bH. Overall, we’d expect aH∗bH=abHaH * bH = abHaH∗bH=abH.

We saw that not all groups satisfy this requirement and so don’t create a group from the cosets. But this requirement is a little hard to work with since it requires us to check if all combinations of the cosets, and when they have different coset representatives. Instead, it’d be great if there was some logically simpler condition that guaranteed aH∗bH=abHaH * bH = abHaH∗bH=abH for free.

Let’s see where our intuition can take us. We know that when we left-multiply some element ggg by any h∈Hh \in Hh∈H, we end up with an element that is ggg-like. You might notice this sounds an awful lot like how the identity works: left-multiplying an element ggg by the identity gives you back ggg. But in hhh’s case, it doesn’t give you back exactly ggg, but something ggg-like. So elements h∈Hh \in Hh∈H are identity-like!

Of course, the usual identity is double-sided. If elements of HHH are truly identity-like, we might expect them to fulfill this duty faithfully. So we’d like that if we right-multiply ggg by some h∈Hh \in Hh∈H, that produces something ggg-like as well. In other words, if we left-multiply ggg by every h∈Hh \in Hh∈H to get all ggg-like elements, they better be the same elements we get from right-multiplying ggg by every h∈Hh \in Hh∈H. In the language of cosets, this condition becomes gH=Hgfor all g∈G.gH = Hg \quad\text{for all } g \in G.gH=Hgfor all g∈G. At a glance, this condition4 seems a lot simpler to work with since we don’t have to check combinations of different cosets. And as a it turns out, the coset operation aH∗bH=abHaH * bH = abHaH∗bH=abH is well-defined if and only if this condition gH=HggH = HggH=Hg for all g∈Gg \in Gg∈G holds5.

Because of their importance, we call subgroups that satisfy this condition normal subgroups in the sense that they are the only subgroups that allow us to form a group out of the cosets. And we call this group of the cosets the quotient group G/H={gH:g∈G}G/H = \{gH : g \in G\}G/H={gH:g∈G} (read GGG mod HHH).

DEFINITION. A subgroup HHH is normal if and only if gH=HggH = HggH=Hg for all g∈Gg \in Gg∈G.

DEFINITION. A quotient group is the set of cosets G/H={gH:g∈G}G/H = \{gH : g \in G\}G/H={gH:g∈G} with the operation ∗*∗ defined by aH∗bH=abHaH * bH = abHaH∗bH=abH for all a,b∈Ga, b \in Ga,b∈G, where HHH acts as the identity element and each coset has an inverse, namely g−1Hg^{-1}Hg−1H.

When I first learned about normal subgroups, they were special subgroups that satisfied gHg−1=HgHg^{-1} = HgHg−1=H. While a logically equivalent statement (and easier to work with when it comes to showing a specific subgroup is normal), it just felt like some random symbolic coincidence. But by observing that HHH ought to act like the identity, the equivalent condition gH=HggH = HggH=Hg became more natural. Further, this intuition of HHH being identity-like extends well to an overarching understanding of quotient groups. You’re breaking down a group by modding/quotienting out the information that you consider to be “just enough like” the identity–i.e. information and detail you don’t care about!

In the first example above, we quotiented Z4\mathbb Z_4Z4​ by H={0,2}H = \{0, 2\}H={0,2}. We considered 222 to be “just enough like” 000 because what differentiates 000 and 222 in Z4\mathbb Z_4Z4​ is irrelevant to us in Z/{0,2}\mathbb Z/\{0, 2\}Z/{0,2}. Both 000 and 222 are even, so all we care about is parity, which is also why the quotient group resembles Z2\mathbb Z_2Z2​. In the second example, we didn’t care about the specific permutations, but again just their parity by considering even-transposition permutations (123)(123)(123) and (132)(132)(132) “just enough like” ()()(). As a result, the quotient group resembles Z2\mathbb Z_2Z2​ with cosets A3A_3A3​ and (12)(12)(12)A3A_3A3​.

The First Isomorphism Theorem

Equipped with these ideas, we can now tackle an important theorem that deals with quotient groups, called the First Isomorphism Theorem. But first, there’s a bit of terminology to introduce. Our process of clumping elements into cosets to map one group to a smaller, less detailed version is actually part of a larger pattern of mapping between one group to another, not-necessarily smaller group. Let’s give this process a name.

If we’re mapping between two groups (G,∗G)(G, *_G)(G,∗G​) and (Gˉ,∗Gˉ)(\bar{G}, *_{\bar{G}})(Gˉ,∗Gˉ​)6 with some function fff, we’d hope that fff preserves the “groupiness” of GGG, namely that fff

  1. Respects the identity. So f(eG)=eGˉf(e_G) = e_{\bar{G}}f(eG​)=eGˉ​.
  2. Respects inverses. So f(g−1)=(f(g))−1f(g^{-1}) = (f(g))^{-1}f(g−1)=(f(g))−1 for all g∈Gg \in Gg∈G. In words, fff maps inverses in GGG to inverses in Gˉ\bar{G}Gˉ.
  3. Respects the operation. So f(a∗Gb)=f(a)∗Gˉf(b)f(a *_G b) = f(a) *_{\bar{G}} f(b)f(a∗G​b)=f(a)∗Gˉ​f(b) for all a,b∈Ga, b \in Ga,b∈G.

It turns out we can derive properties 1 and 2 from 3, so we’ll restrict our attention to 3. Since the functions that satisfy property 3 preserve the “groupiness,” or structure that makes groups interesting, we’ll give them a name: group homomorphisms. Also, since the notation gets cumbersome, let’s drop the group-specific operations from property 3. Just keep in mind they are from different groups.

DEFINITION. A group homomorphism fff between groups GGG and Gˉ\bar{G}Gˉ is a mapping that preserves the group operation, i.e. f(ab)=f(a)f(b)f(ab) = f(a)f(b)f(ab)=f(a)f(b) for all a,b∈Ga, b \in Ga,b∈G

Now given this definition of a group homomorphism and what it does–preserve the operation–you might notice there’s a pretty natural way to understand the “identity-like” intuition from before. If f(b)=ef(b) = ef(b)=e (the identity of Gˉ\bar{G}Gˉ) for some b∈Gb \in Gb∈G, then f(ab)=f(a)f(b)=f(a)e=f(a)f(ba)=f(b)f(a)=ef(a)=f(a). \begin{aligned} f(ab) &= f(a)f(b) = f(a)e = f(a) \\ f(ba) &= f(b)f(a) = ef(a) = f(a). \end{aligned} f(ab)f(ba)​=f(a)f(b)=f(a)e=f(a)=f(b)f(a)=ef(a)=f(a).​ Here, bbb seems to act like the identity under fff exactly because fff maps bbb to the identity. As far as fff is concerned, it is the identity! Now zooming out, we’ll want to talk next about the set of all the elements in GGG that map to the identity, which call the kernel of the group homomorphism fff, denoted ker⁡f\ker fkerf.

DEFINITION. The kernel of a group homomorphism fff are the set of elements of GGG that fff maps to the identity in Gˉ\bar{G}Gˉ, ker⁡f={g∈G:f(g)=e}.\ker f = \{g \in G : f(g) = e\}.kerf={g∈G:f(g)=e}.

<sidetrack> With this definition, allow me to detour a bit to tie this together with the intuition from the previous section. Consider the homomorphism from GGG and its quotient group G/HG/HG/H for some normal subgroup HHH given by ϕ(g)=gH\phi(g) = gHϕ(g)=gH. You might notice this is exactly the function that clumps elements into cosets now in symbols. You can verify it is indeed a homomorphism with the coset operation ∗*∗.

Now observe that for every element h∈Hh \in Hh∈H that ϕ(h)=hH=H\phi(h) = hH = Hϕ(h)=hH=H, and HHH is just the identity element of the quotient group. Conversely, if some element g∈Gg \in Gg∈G maps to the identity, ϕ(g)=H\phi(g) = Hϕ(g)=H, then gH=HgH = HgH=H and g∈Hg \in Hg∈H necessarily. Thus inclusion goes both ways and ker⁡ϕ=H.\ker \phi = H.kerϕ=H. So, a normal subgroup HHH that we hand-wavingly said before contained all the elements that are “identity-like” can now be formalized as exactly the elements that are the identity under ϕ\phiϕ, exactly the kernel of ϕ\phiϕ! In fact, we give this map ϕ\phiϕ that formalizes our natural intuition a name: the natural homomorphism.

Furthermore, since this means every normal subgroup is the kernel of some homomorphism, namely the natural homomorphism, we might wonder if every kernel is a normal subgroup. And indeed it is.7 </sidetrack>

Now we’re ready for the entrée, the First Isomorphism Theorem. Let’s say we have a homomorphism fff from one group GGG to another Gˉ\bar{G}Gˉ. Consider the quotient group of cosets G/ker⁡fG/\ker fG/kerf (which we can create since ker⁡f\ker fkerf is normal): G/ker⁡f={gker⁡f:g∈G}.G/\ker f = \{g\ker f : g \in G\}.G/kerf={gkerf:g∈G}. Since ker⁡f\ker fkerf is the set of elements in GGG that are the identity under fff, you should read each coset element gker⁡fg\ker fgkerf as “the set of elements in GGG that are ggg under fff.”

Given that gker⁡fg \ker fgkerf is all the elements that are ggg under fff, they should behave exactly like ggg under fff. That is, they should all map to where ggg is mapped, namely f(g)f(g)f(g). You might visualize this with colors:

Notice how G/ker⁡fG/\ker fG/kerf splits GGG into clumps of elements that are all the same under fff. Accordingly, these clumps all map to the same element in Gˉ\bar{G}Gˉ. Doesn’t it seem natural that there would be a one-to-one correspondence, and specifically in the context of groups, an isomorphism, between the clumps of colored circles on the left and where they map to colored squares on the right? That’s exactly the First Isomorphism Theorem! It states that,

THEOREM. Let GGG and Gˉ\bar{G}Gˉ be groups and f:G→Gˉf : G \rightarrow \bar{G}f:G→Gˉ be a homomorphism between them. Then G/ker⁡f≅f(G),G / \ker f \cong f(G),G/kerf≅f(G), where f(G)={f(g):g∈G}f(G) = \{f(g) : g \in G\}f(G)={f(g):g∈G} is the image of fff under GGG. Explicitly, the isomorphism is given by the map gker⁡f↦f(g)g\ker f \mapsto f(g)gkerf↦f(g).

Further, notice that because fff is a homomorphism, that all of Gˉ\bar{G}Gˉ need not be hit, or in other words, we could have f(G)≠Gˉf(G) \neq \bar{G}f(G)​=Gˉ. In our picture, there are potentially black squares that are not mapped to, but those don’t concern the First Isomorphism Theorem. All it says is that G/ker⁡fG/\ker fG/kerf is isomorphic onto the image (all the colored squares) of fff under GGG, and that should feel natural!

For the sake of concreteness, let’s apply the First Isomorphism Theorem to an example. Consider the homomorphism f(x)=x mod 4f(x) = x \bmod 4f(x)=xmod4 from Z\mathbb ZZ to Z4\mathbb Z_4Z4​. Then ker⁡f\ker fkerf are all the multiplies of 444, since they are equivalent to 0 mod 40 \bmod 40mod4. Now consider the quotient group Z/ker⁡f=Z/4Z\mathbb Z/\ker f = \mathbb Z/4\mathbb ZZ/kerf=Z/4Z. There are only four cosets in this quotient group, namely 0+4Z,1+4Z,2+4Z0 + 4\mathbb Z, 1+4\mathbb Z, 2+4\mathbb Z0+4Z,1+4Z,2+4Z, and 3+4Z3 + 4\mathbb Z3+4Z since it is cyclic. And of course, by the First Isomorphism Theorem, we know Z/4Z≅f(Z)=Z4\mathbb Z/4\mathbb Z \cong f(\mathbb Z) = \mathbb Z_4Z/4Z≅f(Z)=Z4​ when we map the coset 0+4Z0 + 4\mathbb Z0+4Z to f(0)=0 mod 4=0∈Z4f(0) = 0 \bmod 4 = 0 \in \mathbb Z_4f(0)=0mod4=0∈Z4​, the coset 1+4Z1 + 4\mathbb Z1+4Z to f(1)=1 mod 4=1∈Z4f(1) = 1 \bmod 4 = 1 \in \mathbb Z_4f(1)=1mod4=1∈Z4​, and so on. We’re just mapping the cosets to their representatives  mod 4\bmod 4mod4, i.e. under the homomorphism.

Ideals

Finally, in the spirit of milking this intuition for all it’s worth, we can also motivate ideals–what we get when we generalize the notion of normal subgroups over rings. Suppose we have a ring RRR and we want to break it down into a smaller ring. We can first construct a smaller group by considering the additive abelian group (R,+)(R, +)(R,+). Since it’s abelian, every subgroup III is normal and we know R/I={r+I:r∈R}R/I = \{r + I : r \in R\}R/I={r+I:r∈R} is quotient group. Now, following intuition from earlier, III consists of elements that are identity-like, i.e. 000-like, in (R,+)(R, +)(R,+). Of course, then we would expect that rI⊆IrI \subseteq IrI⊆I and Ir⊆IIr \subseteq IIr⊆I for all r∈Rr \in Rr∈R, since left- and right-multiplication by something 000-like should yield something 000-like. This motivates the ideal, which absorbs multiplication.

And it turns out this ideal condition, that rI,Ir⊆IrI, Ir \subseteq IrI,Ir⊆I for all r∈Rr \in Rr∈R, is exactly necessary and sufficient to define multiplication on the quotient R/IR/IR/I, such that we can create not just a quotient group, but also a quotient ring.

Final Thoughts

Hopefully some of these ideas and intuitions helped you digest what motivates quotient groups and normal subgroups, and how they can be applied and extended. They’ve helped me immensely in internalizing these concepts and how they generalize to other objects, like rings and varieties.
